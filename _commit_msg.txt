now goes FULLSCREEN — the entire screen becomes Jarvis, no borders,

  Fullscreen — entire screen, Jarvis chatbot with text + voice

  - Ambient orb glow pulsing in the background
  - Chat feed: centered, max-width 800px, message bubbles

  - Mode toggle: Voice / Type buttons below the input

    - Voice mode: text input disabled, mic active
